# ☕ «Зернышко» — сайт кофейни

Одностраничный адаптивный сайт для уютной кофейни «Зернышко».  
Проект свёрстан на чистом HTML, CSS и JavaScript без использования фреймворков и сторонних библиотек.
Деплой: 

## 🔍 О проекте

Сайт представляет кофейню, рассказывает о её истории и философии, демонстрирует меню с разбивкой по категориям (кофе, десерты, закуски), показывает отзывы гостей и даёт возможность найти кофейню на карте и связаться с ней.

### Основные возможности:
- **Фиксированный хедер** — навигация всегда перед глазами при прокрутке.
- **Динамическое меню** — переключение между категориями «Кофе», «Десерты», «Закуски» без перезагрузки страницы. Данные хранятся в JS-объекте и рендерятся по единому шаблону.
- **Якорная навигация** — плавный скролл к разделам страницы.
- **Интерактивная карта** — встроенная Яндекс.Карта (или другая) для отображения местоположения.
- **Адаптивная вёрстка** — сайт корректно отображается на десктопах (1200px+), планшетах (до 1230px) и мобильных устройствах (до 820px).
- **Анимации и ховер-эффекты** — плавное увеличение изображений карточек, вращение декоративных элементов, изменение цвета кнопок.

## 🛠 Технологии

- **HTML5** — семантическая разметка.
- **CSS3** — Flexbox, Grid Layout, media-запросы, transitions.
- **JavaScript (ES6+)** — динамический рендер карточек меню, переключение активных категорий.

✨ Особенности реализации

Меню хранится в виде объекта JavaScript, что позволяет легко добавлять и изменять позиции без правки HTML.
Категории меню переключаются сменой CSS-класса active и перерисовкой карточек через JavaScript.
Фиксация хедера реализована через position: fixed с компенсацией отступа через body { padding-top } и html { scroll-padding-top }.
Карта интегрирована отдельным скриптом (coffee.js) и помещена в контейнер с фиксированной высотой.
